Images

HTML represents images through the <img> tag. To insert an image, you must have an image link or the location of the image within your computer.

Once you have this information, the code must be implemented to display an image on your webpage.

This code is shown below:


<img src="name.jpg" alt="alternatetext">

Below, is the same code, though with extra information needed, including the folder name. Without this code, it will not work.


<img src="../foldername/name.jpg" alt="alternatetext">

Tables

The tag th stands for table heading. As the name suggests, it is the heading of the table, meaning that anything you put in a th tag, will act as a heading.

Another aspect in a table is tr, this stands for table row, and whatever is inputed in this will act as a row of a table.

The final portion of a table is table data, or (td). Table data is a singular table cell, including standard cells and header cells.

Common HTML tags

There are a multitude of HTML tags that allow coders to make their own website, these tags can be vital to make a proffesionial and well-constructed webpage, a few of these tags will be taught.

<html>


The HTML tag indicates that it is a HTML document.

<head>


The head tag acts like a container for all the head elements. The head elements consist of a title for the document, scripts, styles, meta information, and more.

<body>

The <body> tag defines the document's body. The body element contains all the content of an HTML document, such as text, images, tables, lists, etc.


<div>

The division tag (<div>), is a section in the HTML document, and is often purposed for CSS or another coding language.

<span>

This tag allows for styling within a tag, such as <p> or <h1>.


This is shown in the following example:


<element> This writing is now <span style="color:orange;"> orange</span></element>

The above structure will be used to show a result from this code:


This writing is now orange

<p>

Paragraphs go into the body of the HTML and can contain <images>, videos and other common HTML tags. It also allows for coders to write paragraphs, like the one you're reading right now!

<header>


Headers, which are found in the body, are represented in the form of h1, h2, h3 ... h6. The larger the number in front of 'h', the smaller the text gets in size. This can give a structure to your webpage, and make it easier to follow your website.

Page Links

When creating more advanced websites with greater content, that require structure, coming from new links and pages, you must be able to link between multiple pages.

To achieve links between HTML pages and CSS pages, you must follow the following line of code:


<link type="text/css" rel="stylesheet" href="../folder/file.css" media="screen and (min-width: 481px)"/>


Purpose
link type="text/css" Dictates the file type (e.g. css)
rel="stylesheet" Indicates the use of CSS for the HTML file
href="../folder/file.css" Shows the folder that the file is found, and is ordered this way
media="screen and (min-width: 481px)" Dictates the size of the screen display

The following code will allow users to navigate to another html page at the click of a mouse:


<a href="pagebeinglinked.html">The Link</a>
