Website overview

This educational website will aid in the learning of HTML, CSS, and the web organisation of web developement.

Internet

The Internet, or Wide World Web (WWW) is a large global system of interconnected networks. Many websites such as this one, can be found on the internet.

In this modern day, it is fundamental to code in Hypertext Markup Language (HTML) to stay relevant, and that is what this website is all about!

Browsers

An internet browser is a software application that allows users to access the WWW. Some important browsers are Google Chrome, Firefox and Safari.

Servers

Servers, are special computers directly connected to the internet. Each internet server has a unique ip adress.

An example of an ip adress is:

Indroduction

The following information will provide you with enough knowledge and education on the topic of HTML and CSS, to commence learning!

HTML

HTML is the underlying basis of website building, it is the foundation to a good and well produced website. Without HTML, you could not make your site!

HTML is quite simple to learn, and is formatted with several tags on the file, which is further discussed in the HTML page.

CSS

CSS is what makes your web-page eye-catching and aesthetically pleasing to users. This language of code styles the foundation (HTML) to give the finishing touches to your file!
